# Tetris Invisible Trainer

Browser-based Tetris trainer focused on invisible practice.

## Play
https://chokotia.github.io/tetris-invisible-trainer/index.html

## Core
- Invisible mode for memory training
- Configurable attack system for garbage practice
- Customizable handling (DAS, ARR, SDF)
- Undo and replay support
- Map code system for board sharing
